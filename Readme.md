## Usage


```bash
curl -XPOST -H 'Content-Type: application/json' -d '{"orderID":333,"toppings":["pepperoni","bacon"],"customerID":1,"size":"L","orderType":"phone","destination":"delivery"}' localhost:3000/pizza/
```
